# Service with Auth

### Start ui projects on http://localhost:3000:

```
npm start
```

### Start simple server on http://localhost:3001:

```
npm run jsonServer
```

### Start project (front+back):

```
npm run concurrently
```

### `node v18.5.0`

## Stack

- typescript
- React/Redux-Toolkit
- MaterialUI
- Axios
- react-router-dom@6
- css-module
